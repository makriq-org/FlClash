# Журнал изменений

Этот файл используется как источник коротких пользовательских release notes для Android-релизов GitHub. Для нового релиза добавляйте секцию `## vX.Y.Z` в начало файла и заполняйте её короткими bullet-пунктами без технических деталей. Подробный технический changelog и заметки по проверке храните отдельно в `docs/releases/vX.Y.Z.md`.

Исторические записи до Android-only поворота могут упоминать desktop-компоненты апстрима. Новые секции в этом файле должны описывать только публично поддерживаемый Android-контур форка.


## v0.9.02-pre1

- Android split tunneling снова корректно применяет списки приложений из профиля, файлов и ссылок.
- Посторонние приложения больше не попадают в правила профиля, если список установленных программ временно недоступен.
- Маршрутизация из профиля теперь сохраняется без подмены глобальными настройками приложения.
