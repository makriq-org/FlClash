# v0.9.02

## Для кого этот релиз

Это стабильный выпуск ветки `v0.9.02` после обкатки предрелизов.

## Что изменилось для пользователя

- Исправлена корректная работа Android split tunneling для списков приложений из файлов и URL.
- Убраны ложные совпадения приложений, если список установленных пакетов временно недоступен.
- Сохранены route-настройки из активного профиля без лишней runtime-перезаписи.

## Технические заметки

- В стабильный тег вошли те же пользовательские исправления, которые уже обкатывались в prerelease-линейке `v0.9.02-pre*`.
- Дополнительные изменения после обкатки касались документации и CI форка, а не пользовательского поведения Android split tunneling.
- Публичный релиз остаётся коротким; все детали сопровождения и релизного процесса вынесены из release body.
