# v0.9.02

## Для кого этот релиз

Стабильный релиз для пользователей Android после обкатки ветки `v0.9.02-pre1`.

## Что изменилось для пользователя

- Исправлена работа Android split tunneling для списков приложений из файлов и URL.
- Убраны ложные совпадения приложений, когда список установленных пакетов временно недоступен.
- Сохранены route-настройки из активного профиля без лишней runtime-перезаписи.

## Что это даёт на практике

- Списки приложений из локальных файлов и удалённых URL снова применяются предсказуемо.
- Android split tunneling устойчивее переживает временные сбои при чтении списка установленных пакетов.
- Профильные route-настройки сохраняются стабильнее и не затираются лишней runtime-логикой.
