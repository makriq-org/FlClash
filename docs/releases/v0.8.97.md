# v0.8.97

## Что изменили технически

### Репозиторий и релизный канал

- Репозиторий переведён на единый основной бранч `main` без оставшихся наследованных веток апстрима.
- Русифицирована публичная поверхность проекта: документация, issue templates, workflow-имена и release-метаданные.
- Релизный канал и самообновление переведены на одну самостоятельную поддерживаемую линию `makriq`.
