# v0.9.01-pre8

## Что изменили технически

### Android split tunneling: без дополнительного окна доступа к приложениям

- Убрано отдельное окно с объяснением перед попыткой получить список установленных приложений.
- Если список приложений недоступен, FlClash по-прежнему не роняет профиль: mask/regex-правила мягко пропускаются, а точные package names продолжают работать.
- Вся логика fallback остаётся клиентской: в Android VPN по-прежнему уходят только точные package names.

### Верификация

- GitHub Actions branch pipeline проверяет `flutter analyze` и `flutter test test/common` для `pre8`; после зелёного branch CI можно не ждать окончания release-матрицы.
