# v0.8.98

## Что изменили технически

### Окно обновления Android

- Окно обновления перестроено вокруг трёх понятных действий: `Позже`, `Пропустить релиз`, `Скачать и установить`.
- Пропуск теперь сохраняется только для конкретного тега релиза и не отключает автопроверку целиком.
- Ручная и автоматическая проверка обновлений выровнены по поведению и больше не дают разных сценариев для одного и того же релиза.
