# v0.9.00-pre1

## Что изменили технически

### Перенос репозитория и контактов

- Публичные ссылки, badges, issue templates и release helper-скрипты переведены на новый адрес `makriq-org/FlClash`.
- Упоминания владельца и packaging-контакты очищены от старого аккаунта `makriq3` и приведены к `makriq`.
- Встроенный repository id приложения синхронизирован с новым репозиторием, чтобы About и release-ссылки больше не вели в старый namespace.

### Предрелизный контур

- Сравнение версий обновлено так, чтобы корректно обрабатывать prerelease-форматы и теги вида `0.9.00-pre1`.
- В настройках добавлен отдельный переключатель для pre-релизного канала обновлений; по умолчанию он выключен.
- Release workflow теперь публикует теги `v*-pre*` как GitHub prerelease с поддерживаемым набором артефактов и `*.sha256`, оставляя Linux RPM только для stable-релизов.
- Документация релизного процесса обновлена под новый репозиторный контекст и отдельный prerelease-поток.
