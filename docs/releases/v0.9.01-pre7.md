# v0.9.01-pre7

## Что изменили технически

### Android split tunneling: подсказка перед системным запросом

- Внутренний confirm-диалог убран: FlClash теперь показывает только короткую русскую подсказку прямо перед попыткой получить список приложений, а дальше уже Android сам решает, нужен ли системный запрос.
- Подсказка показывается только один раз вообще: после первого показа FlClash запоминает именно факт показа, а не доступ к списку приложений.
- Если оболочка не даст доступ к списку приложений, профиль всё равно запустится как обычно, а mask/regex-правила просто будут пропущены.

### Верификация

- GitHub Actions branch pipeline проверяет `flutter analyze` и `flutter test test/common` для `pre7`; после зелёного branch CI можно не ждать окончания release-матрицы.
