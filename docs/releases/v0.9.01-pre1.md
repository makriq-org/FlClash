# v0.9.01-pre1

## Что изменили технически

### Android split tunneling через профиль

- Android split tunneling теперь можно задавать прямо в профиле через `tun.exclude-package` и `tun.include-package`.
- Исключённые через профиль приложения теперь гарантированно обходят VPN целиком, независимо от доменов и IP-адресов.
- Whitelist-режим через профиль тоже поддерживается: через VPN идут только указанные пакеты, а остальные приложения обходят туннель.
- Конфликтные профили с одновременным `tun.include-package` и `tun.exclude-package` теперь отклоняются сразу с понятной ошибкой.

### Документация и верификация

- Добавлена отдельная документация с примерами profile-driven split tunneling для Android.
- Регрессионные тесты дополнены проверками преобразования `include-package` / `exclude-package` в режимы Android `VpnService`.
