# v0.8.96

## Что изменили технически

### Финальная очистка продуктовой идентичности

- Исправлены Android native `JNI` bindings, чтобы переименованный пакет `com.makriq.flclash` корректно работал во всём runtime-контуре.
- Удалены последние старые vendor-метаданные из Windows release manifest, и desktop-бинарники теперь публикуются как `FlClash`.
- Удалены неиспользуемые Firebase-записи из Android version catalog после вырезания Google Services из продуктовой линии.
