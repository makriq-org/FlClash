# v0.9.00

## Что изменили технически

### Обновления и канал релизов

- Репозиторий, ссылки приложения и release-метаданные окончательно переведены на `makriq-org/FlClash` и `makriq`.
- Проверка обновлений получила отдельный pre-релизный переключатель в настройках; по умолчанию он выключен, а stable-канал остаётся безопасным.
- Ручная и автоматическая проверка обновлений теперь используют единый выбор релизного канала и корректно различают stable и prerelease.

### Надёжность релизов

- Сравнение версий обновлено так, чтобы корректно обрабатывать теги вида `0.9.00-pre1` и переходы с prerelease на stable.
- Android updater выбирает совместимый ABI-артефакт и использует `sha256` для проверки скачанного файла.
- Release workflow публикует артефакты и контрольные суммы согласованно для поддерживаемых платформ, а Linux RPM остаётся только для stable-релизов.
