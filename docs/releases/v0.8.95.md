# v0.8.95

## Что изменили технически

### Независимая идентичность продукта и релизов

- Android и desktop bundle identifiers перенесены в независимое пространство имён `makriq`.
- Ссылки на апстрим удалены из интерфейса приложения, packaging metadata и вспомогательных скриптов релиза.
- Packaging metadata для Windows, Linux и macOS обновлены под самостоятельную продуктовую линию.

### Надёжность Android-релизов

- Связка с Firebase / Google Services убрана из Android-сборок, чтобы переименование package id больше не блокировало стабильные релизы.
- Продовые Android-релизы сохранены с подписью, но без старой зависимости на `google-services.json`.
- Оставшиеся Git-hosted зависимости из апстрима переведены на нейтральные пакеты с `pub.dev`, где это возможно.
